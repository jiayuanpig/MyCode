# VScode 快捷键

| 功能               | 快捷键            |
| ------------------ | ----------------- |
| 打开文件夹         | CTRL + K CTRL + O |
| 注释               | CTRL + /          |
| 打开新的编辑器窗口 | CTRL + Shift + N  |
| 关闭编辑器         | CTRL + Shift + W  |
| 删除行             | CTRL + Shift + K  |
| ---                | ----              |
| 格式化文件         | Shift + alt + F   |
