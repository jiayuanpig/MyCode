# GitHub 规范

GitHub是世界上最大的软件远程仓库，是一个面向开源和私有软件项目的托管平台，使用Git做分布式版本控制。
简单的来说，GitHub就是全是界程序员和组织发布程序代码的平台之一，全世界各地的程序员讲自己写的代码上传到这里与大家分享。

## 常用的Git命令

克隆代码：git clone http地址/git地址

拉取代码：git pull

切换分支：git checkout branch_name

查看工作区状态：git status

查看往期提交记录：git log

提交到暂存区：git add

提交到本地仓库：git commit -m

提交到远程仓库：git push
